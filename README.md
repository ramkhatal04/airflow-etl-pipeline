# Airflow ETL Pipeline

## Project Overview

This project demonstrates a production-style ETL pipeline using Apache Airflow deployed on AWS. The pipeline extracts data from an external API, transforms it, stores intermediate data in Amazon S3, loads it into Snowflake, and performs SQL transformations.

## Tech Stack

- Apache Airflow 2.10.5
- Python
- Amazon EC2
- Amazon RDS PostgreSQL
- Amazon S3
- Snowflake
- Git
- GitHub
- GitHub Actions
- Linux (Ubuntu)
